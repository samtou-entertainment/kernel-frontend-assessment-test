# Contact App

A Single Page Application for contacts management. See [a demo here](https://kernel-frontend-assessment-test.vercel.app/)

## Project setup
```
npm install
```

### Compiles and hot-reloads for development

```
npm run dev
```

### Compiles and minifies for production
```
npm run build
```
